# Точка отката (базовая рабочая версия)

**Дата:** 2026-06-15

Состояние, в котором поиск и обогащение работают стабильно (до доработок каталогов и фильтрации).

## Характеристики

- Оригинальный `enrich_event.py` без каталогов
- Оригинальный `run_search.py` без автоочистки
- SearXNG: стандартные движки (без Bing)
- Нет кнопки «Очистить список»
- Свежая БД: 7 seed-запросов, 0 мероприятий

## Восстановление

1. Откатить изменения в `app/` к коммиту/снимку этой даты
2. `docker compose down -v && docker compose up --build -d`
